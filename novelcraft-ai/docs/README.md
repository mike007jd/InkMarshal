# App Documentation Policy

This directory keeps active app-level operating docs only.

- Use the live app docs and code as the source of truth.
- Keep release smoke and ADRs only while they are still actionable.
- Delete stale app-level plans and design notes once they stop matching the repository.
- Do not use this directory as an archive for dead specs, dead plans, or superseded implementation notes.
